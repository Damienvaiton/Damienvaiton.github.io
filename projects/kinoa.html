Mangeuse de Kinoa

  • Tech Stack: Javascript, Discord.js, axios
  • Github URL: Project Link

This project was realized in my personal life

The aim of the project is to create a discord bot that can perform a number of functions related to the management of a discord server (Discord is a VoIP and instant messaging platform, where people can group together on servers to share common interests).

The project revolves around Discord.JS. Discord.JS is a Node.js module that makes it easy to interact with the Discord API. This keeps development simple.

Features:

  • Add a role when a new member arrives on the server
  • Various quizzes, e.g. on flags
