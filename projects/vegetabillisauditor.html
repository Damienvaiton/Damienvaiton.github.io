Vegetabillis Auditor

  • Tech Stack: Java, MVVM, Arduino, Firebase
  • Github URL: Project Link

This project was carried out as part of a semester-long project during the 2nd year of BUT informatique.

The aim of this project is to set up an automated system for real-time measurement of various biological sensors, such as oxygen (O2), carbon dioxide (CO2), luminosity, etc., as part of the practical work carried out by the biology department of the Bourg en Bresse IUT.

The system consists of an Arduino board, which is connected to the various sensors and sends the data to a Firebase database. The data is then displayed in real time on an Android application, enabling teachers and students to follow the progress of measurements.

Features:

  • Real-time data acquisition from biological sensors.
  • Transfer of data to a mobile application for real-time visualization.
  • Data display on the application
  • Application data management (deleting data from a single device or from multiple devices)
