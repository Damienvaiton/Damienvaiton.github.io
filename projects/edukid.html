Edukid

  • Tech Stack: Kotlin, MVVM, Room, LiveData, Coroutines
  • Github URL: Project Link
  • APK: Download

This project was carried out as part of a semester-long project during the 3rd year of BUT informatique.

The project, called Edukid, is an Android mobile application designed for children (aged up to 6) and their parents. Children will be able to learn in a fun way, thanks to multiple mini-games divided into several categories. The application is quite ergonomic for young children.

The application is divided into several categories: animals, colors, shapes, numbers, and letters. Each category has several mini-games that allow children to learn while having fun.
